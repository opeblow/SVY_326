# Security Policy

## Supported Versions

Only the latest version (1.0) is currently supported with security updates.
No earlier versions are maintained.

## Scope

This project is an educational coordinate-rotation calculator for the SVY 326
coursework. It makes no network calls, handles no user data, and has no
external service dependencies. The security risk is inherently low.

## Reporting a Vulnerability

If you discover a security issue, please report it by:

- Opening a [GitHub Issue](https://github.com/opeblow/SVY_326/issues)
- Emailing opeblow2021@gmail.com

Reports will be acknowledged within 5 business days. Given the scope of this
project, a fix or response will typically follow within 2 weeks. Security
reports are treated confidentially until a fix is released.
